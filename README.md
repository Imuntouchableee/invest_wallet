# Invest Wallet - Управление криптопортфелем 🚀

Продвинутое приложение для управления портфелем криптовалют с поддержкой нескольких бирж и красивым киберпанк-интерфейсом.

## ✨ Основные возможности

### 👤 Профиль пользователя
- **Полный профиль** с дополнительной информацией:
  - Полное имя и статус
  - Email, телефон, страна
  - Аватар с выбираемым цветом
  - Статистика портфеля (баланс, активы, сделки)
  - Данные о регистрации и последнем входе

- **Редактирование профиля**:
  - Обновление личной информации
  - Изменение статуса и контактных данных
  - Сохранение предпочтений в БД

### 💰 Управление активами
- Добавление криптовалют в портфель
- Выбор биржи (Bybit, Gate.io, MEXC)
- Расчет количества по сумме инвестиции
- Продажа активов с возвратом баланса
- Удаление активов из портфеля
- Визуализация процента портфеля

### 📊 Аналитика
- Графики цены с историей
- Расчет прибыли/убытка
- Анализ тренда
- Сводка по ценам на всех биржах

### 🔄 Обновление данных
- Автоматическое обновление цен каждые 60 секунд
- Фоновый поток для обновления без блокировки UI
- Сохранение scroll position
- Свежие данные из API

### 🎨 Дизайн
- Киберпанк тема с неоновыми акцентами
- Цветовая схема:
  - Основной цвет: `#00d4ff` (cyan)
  - Зелень: `#00ff88` (neon green)
  - Акцент: `#ff3366` (coral)
  - Темный фон: `#0a0e17`
- Адаптивный интерфейс для разных размеров экрана

## 🗄️ Расширенная БД

### Таблица `users`
```
id (PRIMARY KEY)
name (UNIQUE) - логин
password - пароль
balance - баланс (рубли)
email - почта
phone - телефон
country - страна (по умолчанию: Россия)
full_name - полное имя
avatar_color - цвет аватара (#00d4ff по умолчанию)
bio - статус/описание
registration_date - дата регистрации
last_login - последний вход
total_trades - всего сделок
total_invested - инвестировано
total_profit - общая прибыль
best_asset - лучший актив
notification_enabled - уведомления (1/0)
dark_theme - темная тема (1/0)
two_factor_auth - 2FA (1/0)
```

### Таблица `assets`
```
id (PRIMARY KEY)
user_id (FOREIGN KEY) - пользователь
coin_name - название монеты
quantity - количество
value_rub - стоимость в рублях
purchase_date - дата покупки
purchase_price - цена покупки
```

## 🛠️ Установка

1. Клонируйте репозиторий или распакуйте архив
2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Выполните миграцию БД:
```bash
python migrate_db.py
```

4. Запустите приложение:
```bash
python view.py
```

## 📦 Зависимости

- **Flet** - UI фреймворк (Python)
- **SQLAlchemy** - ORM для БД
- **PostgreSQL** - единая база данных приложения
- **CCXT** - API криптобирж
- **Matplotlib** - графики

## 🔌 API интеграции

Приложение поддерживает следующие биржи через CCXT:
- **Bybit**
- **Gate.io**
- **MEXC**

Данные обновляются в реальном времени с интервалом 60 секунд.

## 📱 Интерфейс

### Главный экран
- Карточка профиля с информацией пользователя (кликабельна для просмотра профиля)
- Статистика портфеля (баланс, портфель, активы, сделки)
- Кнопки действий:
  - Добавить актив
  - Пополнить баланс
  - Сводка по монетам
  - Информация о приложении
  - Выход

- Список активов с:
  - Иконкой и названием
  - Количеством
  - Стоимостью в USD
  - Процентом от портфеля
  - Кнопками анализа, продажи, удаления

### Диалог профиля
- Информация о пользователе
- Контактные данные
- Статистика портфеля
- Информация об аккаунте
- Кнопка редактирования

### Редактирование профиля
- Форма с полями для изменения информации
- Сохранение в БД
- Уведомление об успехе

## 🔐 Безопасность
- Хеширование паролей при регистрации
- Thread-safe работа с БД
- Параметризованные SQL запросы (защита от SQL injection)
